    ####为什么学习JavaScript

    1. HTML定义了网页的内容
    2. CSS描述了网页的布局
    3. JavaScript描述了网页的行为

    ####JavaScript

    1. 直接写入HTML输出流
    2. 对事件的反应
    3. 改变HTML内容
    4. 改变HTML图像
    5. 改变HTML样式

    ####JavaScript用法
    HTML 中的脚本必须位于 标签之间。
    脚本可被放置在 HTML 页面的 和 部分中 。
    script 标签：如需在 HTML 页面中插入 JavaScript，请使用 script 标签。script可以在head中，可以在body中，也可以在外部JavaScript文件中。

    ####JavaScript输出
    windows.alert()：弹出警告框
    操作HTML元素：document.getElementById(id):使用id来标识HTML元素，并用innerHTML来获取或插入元素
    写到HTML文档：document.write()仅仅向文档输出写内容，如果在文档加载完成后执行document.write()，整个HTML文档将被覆盖。
    写到控制台：console.log()在浏览器中显示JavaScript的值。

    ####JavaScript语法
    JavaScript字面量：
    数字(Number)：可以是整数，或者小数，或者是科学计数(e)。
    字符串(String):可以使用单引号或是双引号.
    表达式字面量(+,-,/)
    数组(Array):[1,2,3]
    对象(Objet)字面量:{firstName:”John”,lastName:”Doe”,age:50}
    函数(Function)定义一个函数：function myFunction(a,b){return ab}
    *    JavaScript变量**
    var
    JavaScript操作符
    赋值，算数和位运算符：=，+，-，/
    条件，比较，逻辑运算符：==，！=，<，>
    JavaScript语句
    语句用分号分隔；
    JavaScript关键字
    用于标识要执行的操作
    JavaScript注释
    双斜杠//后面的内容会被省略
    JavaScrip数据类型
    数字
    字符串
    数组
    对象
    数据类型的概念
    如果没有数据类型，表达式无法执行
    JavaScript函数
    JavaSript语句可以写在函数内，函数可以重复引用。
    引用一个函数=调用函数(执行函数内的语句)
    JavaScript语句大小写
    JavaScript对大小写是敏感的
    JavaScript字符集
    JavaScript使用Unicode字符集，Unicode覆盖了所有的字符，包括标点等字符。

    ####JavaScript语句
    javascript向浏览器发送指令，告诉浏览器该做什么。
    document.getElementById(“demo”).innerHTML=”你好”：向id=”demo”的HTML输出“你好”
    分号：分号用于分隔JavaScript语句
    JavaScript：浏览器按照编写顺序依次执行每条语句。
    JavaScript代码块:可以分批地组合起来。代码块以左花括号开始，以右花括号结束。
    JavaScript语句标识符：通常以一个语句标识符为开始，并执行该语句。
    空格：JavaScript会忽略多余的空格，以提高可读性。
    对代码进行折行：可以使用反斜杠\对代码进行换行。

    ####JavaScript注释
    单行注释*：以//开头
    多行注释：以/*开始，以/*结尾

    ####JavaScript变量
    变量是存储信息的容器

    1. 变量必须以字母开头
    2. 变量也能以$，_开头
    3. 变量名称对大小写敏感

    声明(创建)JavaScript变量
    我们使用var关键字来声明变量：var name;
    变量声明之后，它的值是空的，如果需要赋值，要用=:name=’张三’
    一条语句，多个变量
    你可以在一条语句中对多个变量进行声明
    var name = “zhangsan”,age = 18;
    声明也可以跨多行
    一条语句中声明的多个不可以赋同一个值
    value = undefined
    在计算机程序中，经常会声明无值的变量。未使用值来声明的变量，其值实际上是undefined.
    重新声明JavaScript变量
    如果重新声明JavaScript变量，该变量的值不会丢失。
